Sobre

SphyX é um bot brasileiro open-source com diversas funções de moderação, diversão e muito mais! Feito para ser “tudo em um”, evitando que o servidor tenha diversos bots para cada função. O bot já tem todas as funções que você deseja. Funções como:

  • Autorole
  • Mensagens de bem vindo personalizados
  • Comandos de moderação completos
  • Reactionroles mudar o prefixo do bot
  • e muitas outras funções úteis para o seu servidor!


Como foi feito

O bot foi feito na linguagem Python usando a biblioteca discord.py. O código está disponível no GitHub.
